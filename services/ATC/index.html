Hydra-1 Primary RADAR

Hydra-1 HALOTM is a low cost primary surveillance RADAR with the following features:

  • Re-Purposed, Proven Recreational MARINE RADAR
  • ICAO ASTERIX CAT 08, 21, 48 output.
  • Interacting Multi-Mode (IMM) Tracker that Integrates PSR, ADS-B and SSR

MAAS

Mobile Austere ATC System. There are situations where the capability to rapidly deploy an Air Traffic Control (ATC) System can be critical:

  • Disasters (e.g. Haiti))
  • Backup for failed ATC System
  • Military Exercises in remote austere areas
  • Temporary events such as Air Shows out of austere airports
  • Control of Aerial Resources for fighting wildfires
  • Unmanned Traffic Management (UTM) for Drone/Air Traffic Co- ordination for Beyond Visual Line of Sight (BVLOS)

The smaller the ATC System, the easier its transportation. The smallest System “footprint” is Capella Aerospace’s Mobile Austere ATC System (MAAS).

The System includes:

  • Primary Surveillance Radar (PSR)
  • Secondary Surveillance Radar (SSR)
  • Automatic Dependant Surveillance-Broadcast (ADS- B)
  • Interactive Multi-Model (IMM) tracker with safety alerts
  • Distance Measuring Equipment (DME)
  • VHF/UHF Radios
  • Mobile GSM Base Station
  • S-250/G Military shelter

CAMP

Combined Air / Marine Picture for:

  • Co-operative Situational Information Integration (CSII)
  • Global Maritime Situational Awareness (MSA)
  • Immigration Control

Key Features:

  • Custom Sensor Combining Processor optimises view of sea and air
  • Fuses Automatic Information System (AIS) and RADAR into maritime view
  • Fuses Automatic Dependant Surveillance-Broadcast (ADS-B) and RADAR into air view Remote control of features and functions
  • Optional local display and control

Interfaces to multiple cloud remoting networks:

  • Co-operative Situational Information Integration (CSII) system ICAO ASTERIX (air view)
  • EUNAVOR MED operation Sophia
  • Global Maritime Situational Awareness (MSA)

Hydra-6

HYDRA-6 is the Next Generation Combined ADS-B - SSR Surveillance Sensor based on Static, Circular 6-Sector Antenna.

  • Uses (patent pending) WideAngleMonopulseTM Angle Of Arrival (AOA) Processing for SSR (WAMSSR) and ADS-B
  • Performance as Good as Some MSSRs, but much better than Hog Trough Antenna with Sliding Window SSRTM
  • Uses patented DynamicCalibrationTM, on a Scheduled or Demand basis, to Correct Monopulse Amplitude/Phase Offsets
  • Includes SESAR Enhanced Security for ADS-B: Range by Active Interrogation (RAI)
  • Upgrades SESAR Enhanced Security for ADS-B, by Adding AOA.
  • Future updates to include DME and PSR, using patented technique of using SSR as PSR
  • Very High Availability (Reliability + Maintainability). Lower Acquisition and Operational Cost
  • Lower Acquisition and Operational Cost
  • Lower Site Preparation Costs
  • Can Run on Car Batteries Charged By Solar/Wind
